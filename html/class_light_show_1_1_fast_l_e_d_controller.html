◆ FastLEDController()

LightShow::FastLEDController::FastLEDController ( uint32_t  num)
explicit

Create a FastLED controller_ for NeoPixels.

LIGHTSHOW_FASTLED_DATA_PIN must be defined. For example: #define LIGHTSHOW_FASTLED_DATA_PIN PIN_A1

Creating more than one instance of this class is undefined behavior, though it may be supported by a future version of this library.

HELP WANTED: If someone more experienced with FastLED knows how to implement this in a cleaner way, so that multiple FastLED controllers can co-exist referenced by separate instances of this class, a PR would be most welcome.

Parameters
numthe number of led's to control
